## Laravel Event and Listener Handlers

This project consists of a simple demonstration of Events and Listeners in the Laravel ecosystem, given the context presented by the test.

- From the root directory run composer install.
- You must have a MySQL database running locally
- Update the database details in .env to match your local setup
- Run "php artisan migrate" to setup the database tables
- You can run "php artisan optimize:clear" if you want to clear all cache and optimize the application 
- Run "php artisan test" to run all the tests.
